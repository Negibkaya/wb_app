# WB App

Приложение для управления отгрузками Wildberries. Позволяет загружать базу товаров, формировать списки отгрузок с нумерацией коробок, указывать ШК коробов, сохранять прогресс и экспортировать данные в XLSX (два формата) и PDF.

## Стек технологий

- [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) — сборка и dev-сервер
- [Tailwind CSS 4](https://tailwindcss.com/) — стилизация
- [lucide-react](https://lucide.dev/) — иконки
- [xlsx](https://github.com/SheetJS/sheetjs) — парсинг и генерация Excel файлов
- [jsPDF](https://github.com/parallax/jsPDF) + [jspdf-autotable](https://github.com/nicolevanderhoeven/jspdf-autotable) — экспорт в PDF

## Основной функционал

### 📦 Управление товарами

- **Импорт базы товаров** — загрузка файлов Excel/CSV с колонками: Название, Баркод, Артикул
- **Быстрый поиск** — фильтрация товаров по названию, баркоду или артикулу
- **Управление товарами** — редактирование количества и номера коробки для каждой позиции, замена товара

### 📋 Управление поставками

- **Формирование отгрузок** — добавление товаров в поставку с указанием коробки и количества
- **Нумерация коробок** — автоматический подсчет товаров по коробкам, отображение в виде вертикального списка
- **ШК коробов** — возможность указать штрихкод короба для каждой коробки в разделе «Товаров по коробкам»
- **Локальное сохранение** — все данные (товары, поставка, ШК коробов) хранятся в `localStorage` браузера

### 💾 Экспорт в XLSX — два режима

#### 1. Общая сводка

Сводная таблица с группировкой по баркоду (одинаковые баркоды в разных коробках суммируются):

| Баркод     | Количество |
| ---------- | ---------- |
| 1234567890 | 7          |

#### 2. По коробам (распределение)

Детальная разбивка по коробкам. Если ШК короба указан — подставляется он, иначе — номер коробки:

| Баркод товара | Кол-во товаров | ШК короба | Срок годности |
| ------------- | -------------- | --------- | ------------- |
| 1234567890    | 3              | 1         |               |
| 1234567890    | 4              | 2         |               |

### 📄 Экспорт в PDF

- Красивый отчёт с разбивкой по коробкам
- Статистика по каждой коробке: уникальные баркоды, общее количество
- Таблица с подробной информацией: баркод, артикул, наименование, количество
- Поддержка кириллицы (шрифт Roboto)

### 🔄 Восстановление работы

- **Загрузка ранее экспортированного XLSX** — продолжение заполнения поставки с того же места
- Автоматическое сопоставление баркодов с базой товаров

## Использование

### Для пользователей

1. Откройте приложение в браузере
2. Загрузите файл базы товаров (Excel или CSV)
3. Выбирайте товары и добавляйте их в поставку, указывая количество и номер коробки
4. Укажите ШК коробов в разделе «Товаров по коробкам» (ШК подставятся в xlsx при экспорте «По коробам»)
5. Выберите режим экспорта XLSX: «Общая сводка» или «По коробам»
6. Экспортируйте результат в XLSX или PDF
7. Если нужно продолжить позже — скачайте XLSX файл, а затем загрузите его через раздел «Восстановить поставку»

### Формат файла базы товаров

```
Название,Баркод,Артикул
Товар 1,1234567890,ART-001
Товар 2,9876543210,ART-002
```

## Разработка

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для production
npm run build
```
