# WB App

Приложение для управления отгрузками Wildberries. Позволяет загружать базу товаров, формировать списки отгрузок с нумерацией коробок, сохранять прогресс и экспортировать данные в XLSX и PDF.

## Стек технологий

- [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) — сборка и dev-сервер
- [Tailwind CSS 4](https://tailwindcss.com/) — стилизация
- [lucide-react](https://lucide.dev/) — иконки
- [xlsx](https://github.com/SheetJS/sheetjs) — парсинг и генерация Excel файлов
- [jsPDF](https://github.com/parallax/jsPDF) + [jspdf-autotable](https://github.com/nicolevanderhoeven/jspdf-autotable) — экспорт в PDF

## Основной функционал

### 📦 Управление товарами
- **Импорт базы товаров** — загрузка файлов Excel/CSV с колонками: Название, Баркод, Артикул
- **Быстрый поиск** — фильтрация товаров по названию или баркоду
- **Управление товарами** — редактирование количества и номера коробки для каждой позиции

### 📋 Управление поставками
- **Формирование отгрузок** — добавление товаров в поставку с указанием коробки и количества
- **Нумерация коробок** — автоматический подсчет товаров по коробкам
- **Локальное сохранение** — все данные хранятся в `localStorage` браузера

### 💾 Сохранение и восстановление
- **Экспорт в XLSX** — скачивание поставки с 3 столбцами: Баркод, Количество, Коробка
- **Экспорт в PDF** — красивый отчет с разбивкой по коробкам и статистикой
- **Восстановление работы** — загрузка ранее экспортированного XLSX файла для продолжения заполнения

## Использование

### Для пользователей

1. Откройте приложение в браузере
2. Загрузите файл базы товаров (Excel или CSV)
3. Выбирайте товары и добавляйте их в поставку
4. Экспортируйте результат в XLSX или PDF
5. Если нужно продолжить позже — скачайте XLSX файл, а затем загрузите его через раздел "Восстановить поставку"

### Формат файла базы товаров

```
Название,Баркод,Артикул
Товар 1,1234567890,ART-001
Товар 2,9876543210,ART-002
```

## Разработка

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для production
npm run build
```
